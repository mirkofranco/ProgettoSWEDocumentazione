\chapter{Capitolato scelto C4: MegAlexa}
\section{Descrizione generale}
Il\glossario{capitolato}C4 prevede la realizzazione di un applicativo web o mobile in grado di creare dei\glossario{Workflow}per delle\glossario{skill}che poi verranno integrate ad \textit{Amazon}\glossario{Alexa}.
L'applicativo mobile può essere per smartphone con sistema operativo\glossario{Android}oppure\glossario{iOS}.
\section{Finalit\`a del progetto}
All'utente verranno forniti dei connettori che possono essere inseriti all'interno di un Workflow che poi verrà eseguito tramite controllo vocale.  
Per il controllo vocale verrà utilizzato \emph{Amazon Alexa} la quale sarà collegata ad \textit{Amazon Web}\glossario{Services}per gestire ed elaborare i dati raccolti utilizzando vari servizi come \textit{API}\glossario{Gateway}, \textit{Aurora}\glossario{Serverless}e\glossario{DynamoDB}.
I risultati poi verranno forniti tramite GUI web, o mobile, oppure "a voce" da \emph{Amazon Alexa} stessa.
\section{Tecnologie interessate}
\begin{itemize}
	\item \textbf{Alexa}: assistente vocale di Amazon basato su cloud;
	\item \textbf{Amazon Web Services}: insieme di servizi di cloud computing utili per il progetto tra cui:
	\begin{itemize}
	\item \textbf{API Gateway}: servizio per la creazione, manutenzione e protezione di\glossario{API};
	\item \textbf{Aurora Serverless}: consente di eseguire il codice senza considerare il server;
	\item \textbf{DynamoDB}: database non relazionale per applicazioni che necessitano di elevate prestazioni.
	\end{itemize}
	\item \textbf{Node.js$_{G}$}: piattaforma per il motore Javascript V8;
	\item \textbf{HTML5, CSS3, Javascript$_{G}$, Bootstrap} - tecnologie di base per la gestione di interfaccia web lato client;
	\item \textbf{Swift$_{G}$, Kotlin$_{G}$}: Linguaggi di programmazione rispettivamente per\glossario{Android}e\glossario{iOS}.
\end{itemize}
\section{Conclusione}
Il fatto che il progetto proponga di lavorare con una tecnologia nata di recente come \emph{Amazon Alexa$_{G}$} ha portato il team a scegliere questo\glossario{capitolato}. Inoltre il gruppo ha trovato particolarmente interessante l'idea di sviluppare un applicativo mobile. Infine per lo studio delle tecnologie coinvolte, le quali possono tornare utili in futuro.
